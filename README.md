# Roses Burgers Web App

Una aplicación web moderna y rápida para pedir hamburguesas, inspirada en Roses Burgers.

## Características
- **Menú Visual**: Catálogo con imágenes de alta calidad y descripciones detalladas.
- **Categorías**: Navegación fluida entre Hamburguesas, Acompañamientos, Bebidas y Combos.
- **Carrito de Compras**: Gestión dinámica de pedidos con actualización de subtotales en tiempo real.
- **Integración con WhatsApp**: Envío de pedidos directamente al número del local con formato profesional.
- **Diseño Mobile-First**: Optimizado para dispositivos móviles con una interfaz limpia y "thumb-friendly".

## Tecnologías
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Iconos)

## Desarrollo Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/icardibcmoanda-star/burgers-app.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
